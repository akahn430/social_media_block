# Social Block (Chrome Extension)

A Manifest V3 Chrome extension with a focused sidebar for removing distracting Twitter/X UI elements.

## Current scope (Twitter/X)

- Side panel renders one card per requested Twitter component.
- Hovering a card outlines the matching element(s) on the page.
- Toggle ON = keep visible.
- Toggle OFF = hide the element.
- Toggle state persists in Chrome sync storage.
- Works on both `twitter.com` and `x.com`.

## Included component cards

- Left Bar: Upgrade to Premium+, Today’s News, What’s Happening, Who to Follow
- Feed: Header, Post Box, Post Feed
- Left Nav: X Logo, Home, Search, Notifications, Chat, Grok, Premium, Bookmarks, Articles, Profile, More Menu, Create Post, Profile Menu

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this folder.
5. Open a Twitter/X tab, then click the extension icon to open the side panel.
